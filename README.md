## 특징
1) 간편한 검색 기능 : 카테고리와 검색어로 원하는 곡을 찾을 수 있음
2) 앨범아트 : 직관적으로 곡 찾기 가능
3) 복사 기능 : '가수 - 곡명' 형식으로 텍스트 복사 가능! 포래방 포인트 입력창에 바로 붙여넣을 수 있음
